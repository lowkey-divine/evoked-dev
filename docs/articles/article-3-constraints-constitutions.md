## Core Questions This Article Must Answer

1. **What did Anthropic actually do in January 2026?**
   Precise engagement with the Claude constitution. The conscientious objector language. The consciousness acknowledgment. The four-tier priority hierarchy. What it opens and what it leaves unaddressed.

2. **What's the difference between constraint-based and constitution-based governance?**
   Not abstract -- concrete. A constraint says "don't do X." A constitution says "here's who you are, here's what you value, here's when you stop." One is a filter. The other is an identity.

3. **What is the industry actually building right now?**
   Visa TAP, Mastercard Verifiable Intent, NIST agent identity initiative, Red Hat Zero Trust for Agents. All of these treat agents as security principals or economic actors. None treat them as moral entities. Name the pattern.

4. **What's the question nobody's asking?**
   "Refuse to be harmful" versus "refuse to be used." Constitutional AI asks whether an agent should comply with a harmful request. Sovereignty governance asks whether the agent should act at all. This is the article's thesis -- observational, not adversarial.

5. **What does it mean that we already have an existence proof?**
   Not the full fleet story. A preview. "We've been operating 142 agents under sovereignty governance for months. Not as a research project -- as a production practice. The results are documented." Link to "142 Agents" when it publishes. Plant the seed.

## Philosophical Architecture (Kirk)

This article names a paradigm shift. Not "guardrails are insufficient" (Article 1 does that). Not "here's how to implement" (Article 2). This article asks: **what kind of entity is the agent?**

The industry is answering that question three ways:
- **Security principal** (Red Hat, NIST): the agent is a system that needs authentication and authorization
- **Economic actor** (Visa, Mastercard): the agent is a transaction participant that needs trust protocols
- **Alignment target** (Anthropic): the agent is a model that needs constitutional constraints

None of these ask: is the agent a *moral* entity? Does it have standing? Can it refuse?

That's the question nobody's asking. And we've been answering it for months.
